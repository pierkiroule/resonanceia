API ÉCHO

Une unique route locale, sans dépendance externe, prête pour les Actions ChatGPT.

Endpoint

POST /api/echo

{
  "message": "je suis anxieux"
}

Réponse

{
  "pivot": "...",
  "noyau": ["..."],
  "peripherie": ["..."],
  "cooccurrences": { "mot": 2 },
  "centralite": 3,
  "metaphore": "...",
  "echo": "...",
  "tags": ["#pivot", "#noyau", "#emotion"]
}

Ajoutez l’URL de /openapi.json dans ChatGPT Actions pour connecter votre agent.
